# 🇷🇺 Русская локализация PRISM Live Studio

Неофициальный перевод интерфейса **PRISM Live Studio** на русский язык.

[PRISM Live Studio](https://prismlive.com/) — бесплатное стриминговое приложение от NAVER (аналог OBS Studio), поддерживающее трансляции на YouTube, Twitch, CHZZK и другие платформы.

По умолчанию PRISM поддерживает 7 языков: английский, корейский, японский, испанский, португальский, индонезийский и вьетнамский. Данный проект добавляет **русский** как 8-й язык.

## 🍎 Установка на macOS

### ⚡ Способ 1: Автоматическая установка (рекомендуется)

1. Скачайте или клонируйте этот репозиторий:
   ```bash
   git clone https://github.com/pro100vads-blip/prism-live-studio-ru-mac.git
   cd prism-live-studio-ru-mac
   ```

2. Запустите установочный скрипт:
   ```bash
   bash install-macos.sh
   ```

3. Введите пароль администратора когда будет запрошен (нужен для записи в папку приложения).

4. Готово! Теперь [активируйте русский язык](#-активация-русского-языка).

### 🔧 Способ 2: Ручная установка (пошагово)

Если вы предпочитаете сделать всё вручную:

1. **Скачайте репозиторий** — нажмите зелёную кнопку **Code** → **Download ZIP**, распакуйте.

2. **Откройте содержимое приложения:**
   - Откройте **Finder**
   - Перейдите в папку **Программы** (Applications)
   - Найдите **PRISMLiveStudio**
   - Нажмите **правой кнопкой мыши** → **Показать содержимое пакета** (Show Package Contents)

3. **Перейдите в папку с локализациями:**
   ```
   Contents → Resources → data → prism-studio
   ```

4. **Отредактируйте `locale.ini`:**
   - Откройте файл `locale.ini` текстовым редактором (TextEdit, VS Code и т.д.)
   - Добавьте в конец файла следующие строки:
   ```ini
   [ru-RU]
   # 0x0419
   LID=1049
   Name=Русский
   ```
   - Сохраните файл

   > **Примечание:** Для сохранения файла в системную папку может потребоваться разрешение. Если TextEdit не даёт сохранить — скопируйте файл на рабочий стол, отредактируйте, и скопируйте обратно.

5. **Скопируйте файлы перевода:**
   - Перейдите в подпапку `locale`
   - Скопируйте **все содержимое** папки `locale/` из скачанного репозитория в эту папку
   - Для каждого модуля (Chat, Settings, MainFrame и т.д.) файл `ru-RU.ini` должен оказаться рядом с другими языковыми файлами (`en-US.ini`, `ko-KR.ini` и т.д.)

6. **Структура должна выглядеть так:**
   ```
   PRISMLiveStudio.app/Contents/Resources/data/prism-studio/
   ├── locale.ini                    ← отредактированный (добавлена секция [ru-RU])
   └── locale/
       ├── ru-RU.ini                 ← скопирован
       ├── Chat/
       │   ├── en-US.ini             (уже был)
       │   ├── ko-KR.ini             (уже был)
       │   └── ru-RU.ini             ← скопирован
       ├── Settings/
       │   ├── en-US.ini             (уже был)
       │   └── ru-RU.ini             ← скопирован
       └── ... (и так для всех 32 модулей)
   ```

7. Теперь [активируйте русский язык](#-активация-русского-языка).

### 🗑️ Удаление

Чтобы удалить русскую локализацию:

```bash
bash install-macos.sh --uninstall
```

## 🪟 Установка на Windows

> ⚠️ **Статус: НЕ ТЕСТИРОВАЛОСЬ**
>
> Данная локализация разрабатывалась и тестировалась только на macOS. На Windows она **не проверялась**, но теоретически должна работать аналогичным образом. Если вы попробовали — пожалуйста, создайте Issue и расскажите о результатах!

### Примерные шаги (требуют проверки):

1. Найдите папку установки PRISM Live Studio. Обычно это:
   ```
   C:\Program Files\PRISMLiveStudio\data\prism-studio\
   ```
   или
   ```
   C:\Program Files (x86)\PRISMLiveStudio\data\prism-studio\
   ```

2. Отредактируйте `locale.ini` — добавьте секцию `[ru-RU]` как описано выше.

3. Скопируйте все файлы из папки `locale/` репозитория в папку `locale/` приложения.

4. Перезапустите PRISM Live Studio и выберите русский язык в настройках.

> 💡 **Если вы проверили на Windows** — создайте Issue или Pull Request с подтверждением/исправлениями!

## 🌐 Активация русского языка

После установки файлов:

1. Откройте **PRISM Live Studio**
2. Нажмите на иконку **шестерёнки** (Settings / Настройки) в левом нижнем углу
3. Перейдите на вкладку **General** (Общие)
4. Найдите выпадающий список **Language**
5. Выберите **«Русский»**
6. Появится сообщение о необходимости перезапуска — нажмите **OK**
7. Приложение перезапустится с русским интерфейсом

## ⚠️ Важно знать

- **Обновление PRISM затирает перевод.** При обновлении PRISM Live Studio приложение перезаписывается целиком, включая файлы локализации. После обновления нужно заново запустить `install-macos.sh`.

- **43 строки в корневом файле не переведены.** Для них PRISM отобразит оригинальный английский текст. Это незначительные элементы, не влияющие на общее впечатление.

### 📝 Формат файлов перевода

Файлы используют формат INI:
```ini
Ключ="Значение на русском"
```

- Кодировка: **UTF-8**
- Ключи не переводятся — переводятся только значения в кавычках
- Сохраняйте порядок ключей как в оригинальном `en-US.ini`
